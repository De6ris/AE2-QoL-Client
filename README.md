# AE2 QoL Client

* 灵感来源: EAEP Client
* 项目动机: 原作的1.20.1难产了
* 不fork原项目的原因: 不想接手难产项目, 不想用kotlin, 不想配环境
* 代码: 独立实现
* 模组名缩写: aeqc

---

## 前置依赖

mafglib, ae2, jei

---

## 特性列表

可通过快捷键C+Q配置各功能.

* F键在ae终端搜索jei材料
    * ME终端, ME合成终端, 样板编码终端
    * 样板管理终端
    * 扩展样板管理终端(Extended AE)
    * tom存储
* 中键下单jei材料
* 有无线终端时, 对世界中的方块中键下单(需按住ctrl)
* 在下单失败菜单添加侧边按钮, 允许收藏所有缺失材料
* C键在ae终端清空搜索
