# AE2 QoL Client

* 灵感来源: EAEP Client
* 项目动机: 原作的1.20.1难产了
* 不fork原项目的原因: 不想接手难产项目, 不想用kotlin, 不想配环境
* 代码: 独立实现
* 模组名缩写: aeqc

---

## 前置依赖

mafglib, ae2, jei

---

## 特性列表

可通过快捷键C+Q配置各功能.

* F键在ae终端搜索jei材料, C键清空搜索
    * ME终端, ME合成终端, 样板编码终端
    * 样板管理终端
    * 扩展样板管理终端(Extended AE)
    * tom存储
* 中键下单jei材料
* 有无线终端时, 对世界中的方块中键下单(需按住ctrl)
* 对jei中物品按HOME键取出1个, END键取出1组
* 在下单失败菜单添加侧边按钮, 允许收藏所有缺失材料
* 为样板编写自动创建占位符(假合成)
* 按住ctrl填充样板时, 能跳过合并(用于GT装配线等)
* 自动为样板编码终端补充空白样板
* GT联动功能
    * 编写熔融金属样板时转化为锭
    * 编写热锭样板时转化为锭
